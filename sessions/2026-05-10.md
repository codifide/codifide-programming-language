## What Noema is, as of this session close

A working prototype of a programming language designed for agentic AI.
Canonical form is a typed hypergraph serialized as JSON, with a
deterministic byte form hashed by SHA-256 for content addressing. The
Python reference implementation is authoritative for semantics. A Rust
crate implements the canonical form only (not the interpreter) and
agrees with Python byte-for-byte on every example.
