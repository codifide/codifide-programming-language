## What Noema is, as of this session close

A working prototype of a programming language designed for agentic AI.
Canonical form is a typed hypergraph with **two** deterministic byte
serializations — JSON (v0.1 primary, debuggable) and CBOR per RFC 8949
§4.2 (v0.2 binary, 25-30% smaller, agreed by Python and Rust
byte-for-byte). Both serializations are hashed by SHA-256 for content
addressing, producing disjoint identities by design. The Python
reference implementation is authoritative for semantics. A Rust crate
implements both canonical forms and agrees with Python byte-for-byte on
every example.
